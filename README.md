# 자신의 현재 위치 기록 및 찾기 기능
 
### 주요기능

  1. 자신의 위치를 표시해준다.
  2. 1시간 간격으로 앱이 꺼지더라도 자신의 위치를 기록하여 데이터를 생성시킨다.
  3. 앱을 켰을 때 그 동안의 기록을 보여준다.
